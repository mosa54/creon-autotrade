# Creon Auto Trader

이 저장소는 크레온 플러스를 이용한 PyQt5 기반 자동매매 프로그램 예제입니다.

## 요구 사항
- Windows 운영체제
- [크레온 플러스](http://www.creontrade.com/) 설치 및 로그인
- Python 3.8 이상
- `PyQt5`, `pywin32` 패키지 설치

```bash
pip install PyQt5 pywin32
```

## 실행 방법
```bash
python creon_v9.7.py
```

Windows 환경에서만 동작하며, 크레온 플러스가 실행 중이어야 합니다.
